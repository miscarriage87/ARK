# 🚀 START HERE - ARK Digital Calendar

## Quick Start (30 seconds)

1. **Double-click** `run-simple.bat` in this folder
2. **Wait** for the terminal window to open
3. **Press any key** when prompted
4. **Your browser will open** automatically to http://localhost:8080

That's it! The ARK Digital Calendar frontend is now running locally.

## What You'll See

- **Main Page**: Digital calendar interface
- **Static Content**: Pre-built quotes and themes
- **PWA Features**: Offline support, installable app
- **Responsive Design**: Works on mobile and desktop

**Note**: This is the frontend-only version. For AI-generated quotes and full backend features, the backend server needs dependency fixes.

## Files in This Folder

- `start-dev.bat` - **START HERE** - Launches the entire application
- `check-status.bat` - Check if servers are running
- `QUICKSTART.md` - Detailed setup instructions
- `DEVELOPMENT.md` - Full development documentation
- `USER_GUIDE.md` - How to use the application
- `API_DOCUMENTATION.md` - Backend API reference

## Stopping the Application

Simply close the two terminal windows that opened when you started the app.

## Need Help?

1. Check `QUICKSTART.md` for troubleshooting
2. Run `check-status.bat` to verify servers are running
3. Visit http://localhost:8000/docs for API documentation
